HTML

Introduction

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of websites using a system of tags and elements.

Explain the head and the body tags

Head tag: Contains metadata about the document including title, charset, viewport, links to stylesheets, and scripts. This information is not displayed on the page.

Body tag: Contains all the visible content of the webpage including text, images, links, and other elements that users see and interact with.

Basic Tags

Semantic Tags

  • section: Groups related content together
  • article: Contains independent, self-contained content
  • aside: Contains sidebar or tangential content
  • header: Contains introductory content or navigation
  • footer: Contains footer information like copyright

CSS

Introduction

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and styling of HTML documents. It controls colors, fonts, layouts, spacing, and visual effects.

What is the purpose of CSS?

CSS separates content from presentation, allowing you to style elements consistently across multiple pages and create responsive, visually appealing designs.

How to apply CSS to HTML (3 options)

  1. Inline CSS: Apply styles directly to elements using the style attribute: <p style="color: blue;">
  2. Internal CSS: Write styles in a <style> tag within the head section
  3. External CSS: Link an external stylesheet using <link rel="stylesheet" href="styles.css">

What is a selector and CSS rule?

Selector: Targets which HTML elements to style (e.g., p, .class, #id)

CSS Rule: Consists of a selector and a declaration block with properties and values: selector { property: value; }

Basic CSS

Selectors

  • Tag Selector: Selects all elements of a specific tag (p { color: blue; })
  • Class Selector: Selects elements with a specific class (.highlight { background: yellow; })
  • Tag and Class Selector: Combines tag and class (p.highlight { color: red; })
  • ID Selector: Selects a single element with a specific id (#header { font-size: 24px; })

Class vs ID

Class: Can be used on multiple elements, reusable, used for styling groups of elements

ID: Should be unique to one element, used for specific styling or JavaScript targeting

CSS Box Model

The box model consists of (from inside out): content, padding, border, and margin. Every element has these layers that affect its size and spacing.

Example: A box with 20px padding, 2px border, and 10px margin

Media Queries

Media queries allow you to apply different styles based on device characteristics like screen size. Example: @media (max-width: 768px) { body { font-size: 14px; } }

Layout CSS

Display: Grid

Grid creates a two-dimensional layout system with rows and columns. Use grid-template-columns and grid-template-rows to define the structure.

Display: Flex

Flexbox creates a one-dimensional layout for arranging items in rows or columns. Use flex-direction, justify-content, and align-items to control alignment and spacing.
